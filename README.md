# Masakin App
This project is my learning process about Vuejs

## Get Started
1. Clone project
```
git clone https://github.com/zulkiflijufri/vue-masakin.git
```
2. Enter into the project
```
cd /vue-masakin
```
3. Install dependencies
```
npm install
```
4. Run
```
npm run serve
```
